When you click the button, the value of the property [display] for class container, is modified to block and when you click on X, then it is become none

For the second version we change some stylesheet. For example. div.modal background-color to black with opacity 0.2 and height and width to 100%

Then position to fixed plus top and left to 0 to cover all the page

for animation add @keyframes and for .modal_content set position: relative. If you do not set this, it does not work, because the @keyframes cannot uses top property

In fourth step, we add header and footer to our modal. And for that, a div for header and one for footer is required. Then we can put div.header and div.footer inside the div.content, thus they all three become like a single content. I removed animation just for making the code simple.

In this step (=five) we add animation to the div.content and with a note that it appears from top: -300px to top: 0, thus it does not go to the middle of the page and for that we can add padding-top: 200px inside the div.container.

In step 6, we want the div.content appears from bottom. And for that we can change top: -300 and top: 0 to bottom: -300 and bottom: 0. Since the position is relative it comes up in to the middle of the page. If we change the position to fixed then it only appears at the bottom. And change the width:60% to 100% so that it covers all the page long.
